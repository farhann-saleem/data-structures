# Data Structures Deep Dive

Welcome to the `data-structures` repository, a structured learning environment built around three complementary approaches:

- **First-Principles Reasoning**: Design emerges from the problem itself, not from memorized templates.
- **Socratic Learning**: Questions guide the process and reveal understanding through exploration.
- **Reverse Engineering**: We compare our implementations with Python built-ins to validate behavior and refine design.

---

## What’s Inside

This repository contains handcrafted implementations of essential data structures, supported by practical examples and validation tests.

### Core modules

- `stack.py` — stack implementation
- `queue.py` — queue implementation
- `linkedinlist.py` — singly linked list implementation
- `dynamic_arrays.py` — dynamic array implementation
- `heap.py` — heap operations
- `hashmap.py` — hash map implementation
- `bst.py` — binary search tree
- `graph.py` — graph fundamentals

### Practice and support

- `practice/dynamic_arrays_practice.py` — dynamic array exercises and guided practice
- `dynamic-programming/` — dynamic programming practice code
- `recursion/` — recursion practice examples
- `searching/` — search algorithm exercises
- `sorting/` — sorting algorithm practice
- `tests/` — unit tests and validation scripts

---

## Why This Repository

The repository is designed to support learners who want to understand the principles behind data structures, not just reproduce code.

- Build intuition from the base case upward
- Learn by asking questions, not through rote memorization
- Compare custom behavior with Python built-ins
- Practice with targeted exercises and tests

---

## Learning Path

1. Select a concept: stack, queue, dynamic array, tree, or graph.
2. Explore why the structure works, not just how it is implemented.
3. Implement the design from first principles.
4. Validate behavior against Python built-in structures.
5. Reinforce understanding through tests and exercises.

---

## How to Use

Run the practice examples to explore the implementation details and test behavior:

```bash
python3 practice/dynamic_arrays_practice.py
```

Use the `tests/` folder to verify correctness and compare expected outcomes.

---

## Intended Audience

This repository is useful for learners, interview preparation, and anyone who wants a deeper understanding of data structures through reasoning, questioning, and comparison.
